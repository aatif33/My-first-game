#Stone Paper Scissors Game 

A simple **Stone–Paper–Scissors** web game built using **HTML, CSS, and JavaScript**.  
Play manually against the computer or enable **autoplay mode** for automatic rounds.

**Live Demo:**https://aatif33.github.io/My-first-game/

---

##  Features

-  Manual play: Stone | Paper | Scissors
-  Autoplay mode (plays every second)
-  Score tracking (Wins, Losses, Ties)
-  Persistent score using `localStorage`
-  Reset score with confirmation prompt

---

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
